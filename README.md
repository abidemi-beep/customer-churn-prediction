# Customer Churn Prediction

This project predicts customer churn using machine learning models.

## Models Used
- Random Forest
- Gradient Boosting Machine (GBM)
- XGBoost

## Tools
- Python
- Jupyter Notebook
- Pandas, Scikit-learn

## Results
- Best model: GBM (Tuned)
- ROC-AUC: 0.6609
- Accuracy: 70.26%
